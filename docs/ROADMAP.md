# Roadmap

## v0.2

- Tab titles with git branch (inspired by dotfiles xonsh prompt modules)
- `xdg migrate` with **dry-run only** (no destructive home changes without explicit confirmation)
- Profile import/export JSON

## v0.3

- AeroSpace-aware helper: document “open terminal on workspace 1” integration (no hardcoded window manager)
- Sync profile names with public [dotfiles](https://github.com/shahzebqazi/dotfiles) releases

## v1.0

- Performance benchmarks suite (scrollback, session spawn latency, env merge)
- Optional Ghostty / Terminal.app config import (read-only)

## Explicit non-goals

- iOS / tvOS
- Telemetry or network calls
- Forking Ghostty or replacing Terminal.app rendering stack
